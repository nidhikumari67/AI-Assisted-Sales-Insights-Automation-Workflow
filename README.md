# AI-Assisted Sales Insights Automation

This project automates the generation of daily business insights using **N8N**, **OpenAI API**, and **Google Sheets**.

### 🧠 Project Overview
- Created an automated workflow to analyze daily sales data.
- Used **N8N** to connect Google Sheets and OpenAI API for AI-based data transformation.
- Generated AI summaries and appended insights directly into Google Sheets.
- Reduced manual reporting time by **70%**, improving decision-making speed for business teams.

### ⚙️ Tools & Technologies
- N8N (Workflow Automation)
- OpenAI API
- Google Sheets Integration
- No-Code Automation

### 📊 Key Features
- Automated sales report generation  
- AI-driven insight creation  
- Real-time data updates in Google Sheets  
- Scalable for any dataset format (CSV / Google Sheets)
